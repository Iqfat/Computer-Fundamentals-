Dataset Information

The dataset is loosely based on the shopping details for customers of a few cities. The values vary profoundly which makes it simple and basic to reproduce any kind of visualization based on it. The data has been populated randomly and holds no relation to any real systems data or confidential data. It is particularly useful for those new to data analysis, providing a basic dataset to build simple models and visualizations using tools like Python or Power BI.

This dataset includes 2 files: one contains the entire data for customer shopping while the other file has indexed data based on a matrix according to the cities.

Transforming Raw Data Into Meaningful Insights

Using Microsoft Excel

  1. Open Excel: Start Excel and open a new blank workbook.
  2. Import the CSV File:
    • Go to the Data tab.
    • Click on 'Get External Data', then select 'From Text' (in newer versions, you might find 'From Text/CSV').
    • Browse to your CSV file and select it.
    • Excel will open a wizard or a preview pane where you can specify import settings such as the delimiter (commonly a comma for CSV files).
    • After setting your preferences, click 'Load' or 'Finish' to import the CSV data into the Excel sheet.
  3. Save as Excel Workbook:
    • After the CSV data is loaded, go to File > Save As.
    • Choose the location where you want to save the file.
    • From the 'Save as type' dropdown menu, select 'Excel Workbook (*.xlsx)'.
    • Click 'Save'.

Power BI Dashboard Tutorial

Step 1: Load the Data into Power BI

  1. Open Power BI Desktop.
  2. Select 'Get Data' from the Home ribbon and choose 'Excel'.
  3. Navigate to your file, select it, and then load it.
  4. Choose the sheet that contains your data and click 'Load'.
